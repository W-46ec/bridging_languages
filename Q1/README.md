# Question 1

## Compilation & Execution

We implemented Quick sort in C++, Merge sort in JavaScript and Heap sort in python.

- Quick sort
```
g++ -o quicksort quicksort.cpp
quicksort < inputFile
```

- Merge sort
```
node mergesort.js < inputFile
```

- Heap sort
```
python3 heapsort.py < inputFile
```
